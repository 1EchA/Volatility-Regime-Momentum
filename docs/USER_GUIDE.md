# 📖 用户使用指南

> A股量化策略研究平台详细操作手册

## 🎯 界面总览

本平台采用**标签页设计**，提供4个核心功能模块：

### 📊 **总揽** - 策略表现概览
**用途**: 查看最新策略分析结果，快速了解整体表现

**主要功能**:
- 📈 关键指标展示 (IC均值、信息比率、年化收益、最大回撤)
- 🔥 参数热力图 (可视化不同参数组合的表现)
- 📋 策略表现排行榜
- ⚡ 一键刷新最新结果

**操作步骤**:
1. 页面自动加载最新分析结果
2. 查看关键指标卡片
3. 点击热力图查看参数敏感性
4. 使用"刷新"按钮获取最新数据

---

### 🔍 **网格分析** - 参数优化中心
**用途**: 进行多维参数搜索和策略优化

**核心功能**:
- 🎛️ 参数网格配置
- 📊 换手率策略网格
- 💰 成本敏感性分析
- 🔒 稳健性验证

**详细操作**:

#### 🎯 基础参数设置
```
EMA参数范围: 3-6, 12-18 (短期-长期动量)
Top/Bottom分位: 30-45 (选股数量控制)
交易成本: 0.03%-0.05% (现实成本假设)
```

#### 📊 热力图解读
- **颜色深浅**: 策略表现优劣 (越亮越好)
- **X轴**: 通常为Top_N参数
- **Y轴**: 通常为Bottom_N参数
- **点击交互**: 点击热力图单元格查看详细参数

#### 🔧 高级功能 (展开使用)
- **换手率约束**: 控制策略交易频率
- **制度敏感性**: 不同市场环境下的表现
- **成本冲击分析**: 交易成本对收益的影响

---

### 🎯 **个股查询** - 单股票深度分析
**用途**: 深入分析单个股票的表现和信号

**操作流程**:

#### 1️⃣ 股票选择
- **下拉选择**: 从500+股票池中选择 (格式: 代码(名称))
- **手动输入**: 支持6位股票代码输入
- **股票池统计**: 查看当前股票池概览信息

#### 2️⃣ 参数配置
```
Top N选股: 35 (建议值)
Bottom N选股: 35 (建议值)
预测文件: 自动选择最新文件
```

#### 3️⃣ 分析结果
- **📈 价格走势图**: 股价与技术指标
- **🎯 策略信号**: 买入/卖出信号时点
- **📊 波动率制度**: 不同市场环境着色
- **📋 详细统计**: 收益率、夏普比率等指标

#### 💡 **简洁模式 vs 高级模式**
- **简洁模式**: 核心图表+关键指标
- **高级模式**: 增加行业分位曲线、进出场事件表等

---

### ⚙️ **执行画像** - 策略执行分析
**用途**: 分析策略的实际执行情况和优化建议

**核心指标**:
- **📊 换手率分布**: 不同参数下的交易频率
- **💰 成本影响**: 交易成本对净收益的影响
- **📈 收益分解**: 多头、空头、多空组合收益拆解
- **🔄 调仓频率**: 策略调仓节奏分析

**使用建议**:
1. 关注换手率-收益的权衡关系
2. 根据成本冲击调整参数
3. 监控策略容量约束
4. 优化调仓频率

---

## 🔧 常用操作技巧

### 📱 界面导航
- **标签切换**: 点击顶部标签在不同模块间切换
- **侧边栏**: 左侧参数设置区域
- **展开折叠**: 高级功能默认折叠，需要时展开

### 🎛️ 参数调优建议
- **初次使用**: 建议使用默认参数
- **策略优化**: 先看总揽确定大方向，再用网格分析精调
- **风险控制**: 关注最大回撤指标，避免过度优化

### 📊 结果解读
- **IC值**: >0.05为较好，>0.1为优秀
- **IR值**: >0.5为可接受，>1.0为优秀
- **年化收益**: 考虑回撤后的风险调整收益
- **换手率**: 控制在合理范围内(通常<100%)

### ⚡ 性能优化
- **缓存机制**: 相同参数会自动使用缓存结果
- **分块加载**: 大文件自动分块处理
- **后台处理**: 长时间计算在后台进行

## ❗ 注意事项

### 🚨 数据注意事项
- 确保数据文件完整性
- 注意缺失值处理
- 关注数据更新频率

### 💻 性能注意事项
- 大参数网格搜索耗时较长
- 建议先小范围测试再扩大
- 监控内存使用情况

### 📈 分析注意事项
- 避免数据窥探偏差
- 注意样本外表现验证
- 考虑交易成本的现实影响

## 🆘 故障排除

### 常见问题解决

**Q: 页面加载缓慢或卡死**
```
解决方案:
1. 刷新页面重试
2. 检查数据文件大小
3. 降低参数搜索范围
4. 重启Streamlit应用
```

**Q: 图表无法显示**
```
解决方案:
1. 检查数据文件路径
2. 确认结果文件存在
3. 查看浏览器控制台错误
4. 尝试切换浏览器
```

**Q: 参数设置无效**
```
解决方案:
1. 确认参数范围合理
2. 检查文件格式正确
3. 重新选择预测文件
4. 清除浏览器缓存
```
